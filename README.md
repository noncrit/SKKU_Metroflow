# MetroFlow
지하철 혼잡도 정보 제공 서비스
## 프로젝트 소개
- 지하철 혼잡도 정보를 제공하는 것을 목표로 한 프로젝트입니다.
- 공공 데이터 포털 API를 데이터를 이용해 혼잡도 예측값을 제공합니다.
